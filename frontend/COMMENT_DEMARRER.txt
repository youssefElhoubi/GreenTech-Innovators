📋 PROBLÈME DÉTECTÉ
───────────────────────────────────────────────────────────────────

Votre npm n'installe pas correctement toutes les dépendances (seulement 8-12
packages au lieu de 200+). Ceci est un problème connu avec certaines configs npm.


✅ SOLUTION RAPIDE - UTILISER NPX
───────────────────────────────────────────────────────────────────

NPX permet d'exécuter Vite SANS l'installer localement !


🚀 LANCEMENT EN 3 ÉTAPES :

1. Ouvrir PowerShell dans le dossier Frontend

2. Taper UNE de ces commandes :
   
   Option A (PowerShell) :
   .\start-npx.ps1
   
   Option B (Command Prompt) :
   start-npx.bat
   
   Option C (Manuel) :
   npx vite

3. Ouvrir votre navigateur sur :
   http://localhost:5173


📱 URLS DE L'APPLICATION
───────────────────────────────────────────────────────────────────

• Avec NPX     : http://localhost:5173  ← UTILISEZ CELLE-CI
• Avec npm run : http://localhost:3000  (si npm install fonctionnait)


🎯 SCRIPTS DISPONIBLES
───────────────────────────────────────────────────────────────────

✅ start-npx.ps1  → Script PowerShell (RECOMMANDÉ)
✅ start-npx.bat  → Script Command Prompt
⚠️  start.ps1     → Nécessite npm install (ne fonctionne pas chez vous)
⚠️  start.bat     → Nécessite npm install (ne fonctionne pas chez vous)


🔧 AUTRES COMMANDES UTILES
───────────────────────────────────────────────────────────────────

• Changer de port :
  npx vite --port 3000

• Ouvrir automatiquement le navigateur :
  npx vite --open

• Build pour production :
  npx vite build

• Arrêter le serveur :
  Ctrl + C dans le terminal

🐛 EN CAS DE PROBLÈME
───────────────────────────────────────────────────────────────────

Erreur "port 5173 already in use" ?
→ npx vite --port 3000

Scripts PowerShell bloqués ?
→ Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
→ .\start-npx.ps1

NPX ne fonctionne pas ?
→ Vérifiez votre connexion internet (npx télécharge Vite)
→ npm cache clean --force
→ npx clear-npx-cache


✨ POURQUOI NPX FONCTIONNE ?
───────────────────────────────────────────────────────────────────

NPX télécharge Vite dans un cache temporaire et l'exécute directement.
Pas besoin que npm install fonctionne correctement !

Avantages :
✅ Pas d'installation locale
✅ Toujours la dernière version
✅ Contourne les problèmes npm
✅ Rapide à démarrer
